# adam-mcp

FreeCAD MCP Server

## Installation

1. Install FreeCAD:
   - macOS: `brew install freecad`
   - Linux: `sudo apt-get install freecad` (or equivalent for your distribution)

2. Install Python dependencies:
   ```bash
   uv sync
   ```

## Dependencies

- FreeCAD (installed separately)
- fastmcp >= 1.0.0
- pydantic >= 2.0.0
